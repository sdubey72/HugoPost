Bar graph and Histogram

Here it is ! But it looks very cluttered . Lets see if we can change the axis and make it more readable.

data2<-Teams%>%
  filter(yearID==1980)%>%
  select(name,HR)%>%
  arrange(HR)

data2$name<-factor(data2$name,levels = data2$name)

Now lets try to do some histogram.

df<-Master%>%
  filter(!is.na(weight))%>%
  select(weight)
ggplot()+
  geom_histogram(data=df,aes(x=weight),color="white",fill="blue")+
  ggtitle("baseball players weights")
